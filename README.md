Motor Insurance Management System
🚗 Motor Insurance Management System (MySQL Project)

A complete **Motor Insurance Management System** developed using **MySQL**. This project demonstrates database design, SQL programming, and advanced MySQL concepts as part of a Database Management System (DBMS) academic project.

---
 📌 Project Overview

The Motor Insurance Management System manages the complete insurance lifecycle, including:

* Customer Management
* Vehicle Registration
* Insurance Companies
* Insurance Plans
* Policy Management
* Premium Calculation
* Claims Management
* Payments
* Renewals
* Reports

This project covers almost every important MySQL topic from beginner to advanced level.

---
🎯 Objectives

* Design a real-world relational database.
* Implement normalization and constraints.
* Perform CRUD operations.
* Demonstrate advanced SQL concepts.
* Generate reports using SQL queries.
* Learn MySQL programming objects.

🚀 How to Run

1. Install MySQL Server.
2. Open MySQL Workbench.
3. Create a new database.
4. Execute `create_tables.sql`.
5. Execute `insert_data.sql`.
6. Execute the remaining SQL files in order:

   * Views
   * Stored Procedures
   * Functions
   * Triggers
   * Transactions
   * CTE
   * Window Functions
   * Indexes
   * Reports
